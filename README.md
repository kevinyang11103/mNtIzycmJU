# 项目概述 python279

本项目是一款基于协同过滤算法的音乐推荐系统，旨在为用户提供个性化音乐推荐服务，同时为音乐管理员提供一个全面的音乐管理解决方案。

## 技术栈

### 前端技术
- Html
- Css
- Js
- Vue2.0
- Element-ui

### 后端技术
- Python3.7
- Django2.0

## 运行环境
- 开发工具：PyCharm + VSCODE
- 数据库：MySQL5.7（最低版本要求）
- 数据库管理工具：Navicat10+
- 前端Nodejs：14
- 浏览器：谷歌浏览器

## 功能模块
- 用户管理
- 音乐分类管理
- 音乐信息管理
- 歌曲数据管理
- 系统管理
- 我的信息

## 系统角色
- 音乐管理员
- 普通用户

## 部署步骤
1. 环境准备：确保已安装Python3.7、MySQL5.7、Nodejs14等依赖环境。
2. 数据库配置：使用Navicat10+导入数据库文件。
3. 代码部署：将源码放置在指定目录，使用PyCharm和VSCODE进行项目配置。
4. 前后端运行：分别启动后端和前端服务。

## 目录结构
```plaintext
/
├── frontend/          # 前端代码目录
├── backend/           # 后端代码目录
├── database/          # 数据库文件目录
├── design_doc/        # 设计文档目录
```

## 核心亮点
- 基于协同过滤算法，实现个性化音乐推荐。
- 使用Vue2.0和Element-ui构建用户友好的前端界面。
- 提供全面的音乐管理功能，满足音乐管理员需求。
